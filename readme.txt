=====职务说明=====

程序：
研究制作作品的免CD补丁，引擎的解包与封装，文件的批处理；改进翻译系统，未来如果会建站或搭论坛的话也有劳程序君了

美工：
对带日语的CG进行修图嵌字，为视频添加特效字幕，汉化作品的界面。如果有漫画本子的任务美工君就是主力了

移植：
尝试将汉化完成的作品进行移植，实验性项目，不要求一定完成

翻译：
大家懂的

校对：
快速准确的找出译文的错误和不足并修改，并将比较重要的错误告知翻译君。决定一份文本是回炉重练还是改后采用。同时我希望校对君能够在群内担当叫兽，指点翻译君

中文校对（润色）：
因为我希望文本尽量忠实原文，所以不需要把语句改的过于俏皮和华丽。主要任务是找出文本中汉字的错误、不通顺的语句和不恰当的语气


=====翻译系统=====

网址：219.218.109.221/galkanka/login.php

翻译、校对、润色君请单独联系群管理员，告诉他你想要的账号和密码与职务，管理员将把你添加进数据库，之后只需要选对应身份登录翻译系统即可开始工作。

点击领取来领取文本，该文本将自动标注为你所进行。完成工作后上传。注意自行备份原文件。

如出现其它问题请联系管理员。


=====翻译须知=====

采用覆盖修改方式，请注意自行备份原文件。

文本绝对不可多行、少行、串行，如发现有缺行痕迹请立刻告知管理。

人名行不可超字数。最后一行“指定されたラベルはつかりませんでした”不需翻译。

文中每句话首尾的繁体引号保留。

遇到不会翻或没把握的地方先多问问，仍然未解决的话在当前句首添加一个“[MARK]”标记用来提醒校对。


校对、润色对文本的修改同样遵循以上。
